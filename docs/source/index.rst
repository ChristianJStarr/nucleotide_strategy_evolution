Features
--------

* **Rich Genetic Representation:** Encoding complex rules and adaptive behaviors directly into DNA
* **Multi-Objective Optimization:** Balancing profitability and risk metrics
* **Diversity Preservation:** Using novelty search and MAP-Elites to encourage exploration
* **Rigorous Validation:** Walk-Forward Optimization and Purged K-Fold Cross-Validation
